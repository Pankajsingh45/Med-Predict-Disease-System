# 🏥 MedPredict AI — Early Disease Detection System

A machine learning powered web application that predicts diseases 
from user-reported symptoms using a Random Forest classification model.

## 📁 GitHub Repository
[View Code](https://github.com/Pankajsingh45/Med-Predict-Disease-System)

## ⚙️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python, Flask |
| ML Model | scikit-learn, Random Forest |
| Data Processing | Pandas, NumPy |
| Database | SQLAlchemy |
| Frontend | HTML5, CSS3, JavaScript |

## ✨ Features

- 🔬 Predicts top-3 possible diseases with probability scores
- 🧠 Random Forest classification model for accurate predictions
- 🔍 Fuzzy symptom matching for better user input flexibility
- ⚡ Real-time prediction REST API
- 📱 Fully responsive cross-device UI
- 🗄️ SQLAlchemy powered database integration

## 🚀 Getting Started

```bash
git clone https://github.com/Pankajsingh45/Med-Predict-Disease-System
cd Med-Predict-Disease-System
pip install -r requirements.txt
python app.py
```
